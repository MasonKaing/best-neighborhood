Datasets used:
https://data.wprdc.org/dataset/311-data/resource/76fda9d0-69be-4dd5-8108-0de7907fc5a4 <br>
This is a dataset with the amount of 311 service requests in the City of Pittsburgh. This is the dataset used to determine cleanliness.

https://data.wprdc.org/dataset/pbp_arrest_data_2024_2025  <br>
This is a dataset that keeps track of police arrests. This is used for our police arrest count metric.

https://data.wprdc.org/dataset/emissions-inventory/resource/1ab77bb5-5684-430e-bdda-fb401167fb6a  <br>
This is a dataset that tracks air quality emissions. This is used for our air quality metric.

Description of Project: <br>
This repository has notebooks for our personal sub-metrics and the final notebook with all of our information combined to find the best neighborhood using our three sub-metrics.
